HTML

Type: Markup language

Extension: .html, .htm

Developed by: W3C & WHATWG

Initial release: 1993

Standard: HTML Living Standard

Website: w3.org/html

Overview

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It describes the structure of a webpage semantically and originally included cues for the appearance of the document.

History

HTML was created by Tim Berners-Lee in 1991, but "HTML 2.0", published in 1995, was the first standard HTML specification. HTML is maintained by a consortium of the W3C and WHATWG.

Features

  • Elements and tags
  • Attributes
  • Headings, paragraphs, and text formatting
  • Links, images, and multimedia
  • Forms and input elements
  • Semantic elements

Example


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Example HTML</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is an example of HTML code.</p>
</body>
</html>
